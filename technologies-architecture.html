Technologies and Architecture

CloudFlare - CDN, DNS & Proxy Server

CloudFlare is an American company that offers a reverse proxy solution which can be used to speed up your website by using its content delivery network and protect it against hacks, SQL injections. In addition to that, it also offers attractive features such as free SSL certificate, DoS protection, load balancing and more. We are using CloudFlare at Passfree to manage our DNS records and use their SSL certificate(s).

Nginx - Web Server

Nginx is an event-based web-server known for its stability and high performance. It is used to serve web content such as HTML and is often used in pair with PHP to serve dynamic content. We'll be using Nginx at Passfree to serve our web-application to the users. Every request is first processed by Nginx and then sent to PHP for processing and then sent back to the user.

Izaro - MVC Framework

Model-View-Controller (MVC) is an architectural design used in programming web-applications to separate the business logic and presentation layer from each other. It is used to separate the HTML content (view) with data processing unit (model) and the link between the view and models (controller). MVC enables modularity, clean and extensible code. At Passfree, we are using a small MVC framework called Izaro to build our application. Some of its features include template engine, automatic URL routing, support for authentication, etc. We have decided to take Izaro over Laravel in this case for its simplicity and friendliness with beginners.

MySQL - Database Server

MySQL is an opensource RDBMS software based on the SQL language. Owing to its high performance and ease of use, MySQL is now a widely used database server that is coupled with PHP. In our case, MySQL would be used to store user accounts, their encrypted passwords and 2FA keys.

GitHub - Code Collaboration

GitHub is a Git hosting service with added features. GitHub provides a graphical user interface for Git utility. It also provides code collaboration tools and support for project documentation, wikis, etc. We are using GitHub to collaborate on our project.

PHPStorm - IDE

PHPStorm, just like VSCode, is a popular text-editor and development environment specially made for PHP. It supports error prevention, code analysis and code refactoring. In addition to that, it also supports PHP frameworks such as Laravel, Symphony, CakePHP, etc. We are using PHPStorm to write our source code.
